Purpose: Enrich company data using YTJ API and xAI analysis

Features:

  • YTJ API integration for official company data
  • NACE code to industry mapping
  • Growth stage inference (pre-seed to scale-up)

2. xAI Service

File: backend/services/xai_service.py

Purpose: Generate AI-powered company descriptions with web search

Features:

  • Real-time web search for company intelligence
  • Market size estimation
  • Citation tracking for transparency
  • Finnish to English translation
  • Fallback mechanisms for API failures

3. Funding Discovery Engine

Files: funding_discovery.py, xai_funding_discovery.py

Purpose: Discover current funding programs from multiple sources

Features:

  • Web scrapers for Business Finland, ELY, Finnvera
  • xAI-powered dynamic URL discovery
  • Global rate limiting (2 req/sec per domain)
  • Response caching (24-hour TTL)
  • Fallback to hardcoded programs

4. Matching Engine

File: backend/services/matching_engine.py

Purpose: Score and rank funding programs for each company

Scoring Algorithm:

  • Industry alignment: 30%
  • Geographic eligibility: 25%
  • Company size fit: 20%
  • Funding amount match: 15%
  • Application timing: 10%

Performance Optimizations

  • Parallel Processing: Company description and funding discovery run concurrently (Promise.all)
  • Multi-level Caching:
    • Redis for API responses (xAI, YTJ) - 1 hour TTL
    • File-based cache for scraping results - 24 hour TTL
    • In-memory cache for frequently accessed data
  • Rate Limiting: Global rate limiter prevents website blocking (2 req/sec per domain)
  • Connection Pooling: Reuse HTTP connections for faster requests
  • Lazy Loading: Frontend loads results progressively

Security & Privacy

  • CORS: Restrict API access to frontend origin only
  • Input Validation: Pydantic models validate all incoming data
  • Rate Limiting: Prevent abuse (100 requests/hour per IP)
  • Data Encryption: HTTPS in production, encrypted cache storage
  • GDPR Compliance: No personal data storage, company data from public sources only
  • API Key Security: xAI keys stored in environment variables, never committed to Git
